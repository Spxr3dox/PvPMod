
Kopieren

================================================================
                     LegitMod  v1.0.0
          Minecraft 1.21.4 · Fabric Loader 0.16.9+
================================================================
 
Легкий клієнтський мод для PvP з набором функцій для легіту.
Не відображається в налаштуваннях керування — всі клавіші
жорстко прошиті і не змінюються.
 
================================================================
  ВСТАНОВЛЕННЯ
================================================================
 
1. Встанови Fabric Loader 0.16.9+ для Minecraft 1.21.4
2. Встанови Fabric API
3. Скопіюй legitmod-1.0.0.jar у папку .minecraft/mods/
4. Запускай гру
 
================================================================
  ЗБІРКА З ВИХІДНОГО КОДУ
================================================================
 
  .\gradlew.bat build          (Windows)
  ./gradlew build              (Linux / macOS)
 
Готовий .jar знаходиться в:  build/libs/legitmod-1.0.0.jar
 
================================================================
  КЛАВІШІ КЕРУВАННЯ
================================================================
 
  Y       — TriggerBot (вкл/викл)
  G       — AimBot (вкл/викл)
  R       — Комбо AimBot + TriggerBot:
              · Обидва вимкнені → увімкнути обидва
              · AimBot вмкн, TriggerBot вимкн → увімкнути TriggerBot
              · Обидва увімкнені → вимкнути обидва
  Z       — TargetStrafe (вкл/викл)
  B       — Hitbox (вкл/викл)
  =       — Hitbox збільшити (+0.05 блоків)
  -       — Hitbox зменшити (-0.05 блоків)
  [       — ESP (вкл/викл)
  I       — AutoSprint (вкл/викл)
  H       — ShiftTap (вкл/викл)
  N       — NameProtect (вкл/викл)
  V       — ElytraSwap (одягти/зняти елітру)
  →       — AutoAuth (вкл/викл)
  ↑       — AutoReg (вкл/викл)
  ←       — UnHook — вимкнути і заховати небезпечні функції
             (TriggerBot, AimBot, Hitbox, ESP, TargetStrafe)
             Повторне натискання — повернути все назад
 
================================================================
  ФУНКЦІЇ — ДЕТАЛЬНИЙ ОПИС
================================================================
 
----------------------------------------------------------------
  COMBAT
----------------------------------------------------------------
 
TriggerBot  [Y]
  Автоматично атакує будь-яку живу сутність на яку наводишся
  з крутаймером. Атакує рівно в момент повного відкату зброї
  з відхиленням 1–10 мс (виглядає як ручні удари).
  · Вимога: тримати меч (дерев'яний–незеритовий)
  · Дальність: 3.0 блоки
  · Синхронізується з Hitbox
 
AimBot  [G]
  Плавно наводить прицілювання на нагрудник найближчого
  гравця з найменшим HP у радіусі 6 блоків.
  · Плавність залежить від дистанції: близько — повільніше,
    далеко — швидший snap
  · Блокує ціль після першого наведення; змінює тільки якщо
    ціль пішла >20 блоків або вийшла з гри
  · Мікро-відхилення кожні 3-6 тіків для маскування
  · Перезапустити (вибрати нову ціль): вимкнути і увімкнути G
 
TargetStrafe  [Z]
  Незалежна функція — рухається навколо обраного гравця
  зі швидкістю спідів, автоматично б'є крітами і наводиться,
  при цьому ти можеш дивитись куди хочеш (фрі вид).
 
  Як це працює:
  · Вибирає гравця з найменшим HP у радіусі 24 блоків
  · Кружляє навколо нього на відстані 0.5 блока
  · Автоматично стрибає перед ударом для крита
  · Голова і тіло наводяться на ціль (інші бачать коректно)
  · Твоя камера залишається вільною — можна дивитись куди хочеш
  · Вбудований тригербот: б'є при 98.5% відкату зброї
  · Голова рухається вгору-вниз при ударі (анімація)
  · Швидкість зростає якщо не може дотягнутись до цілі
  · Показує красивий 3D маркер на ціль:
      - Обертове кільце на рівні грудей
      - Кутові дужки навколо гравця (мигають)
      - Вертикальний промінь над головою
  · Показує BPS (блоків/сек) в кутку екрану
  · Працює навіть при відкритому інвентарі
  · Зупиняється тільки якщо ціль виходить з серверу або
    відходить >24 блоків (тоді знаходить нову)
  · Для зміни цілі: вимкнути і увімкнути Z
 
----------------------------------------------------------------
  MOVEMENT
----------------------------------------------------------------
 
AutoSprint  [I]
  Автоматичний спрінт при натисканні W + вбудований WTap:
  при ударі зупиняє спрінт на 3 тіки (зменшує відкидання
  противника), потім відновлює.
  · Вимога: їжа > 6 одиниць, не присідати, не блокувати
 
ShiftTap  [H]
  При кожному ударі натискає Shift на 2-3 тіки (рандомно).
  Додає невеликий downward momentum — трюк для PvP.
  Не заважає AutoSprint.
 
ElytraSwap  [V]
  Миттєво міняє нагрудник на елітру і назад.
  · Перше натискання V: знімає нагрудник → одягає елітру
  · Друге натискання V: повертає елітру → одягає нагрудник
  · Якщо нагрудника немає — просто одягає/знімає елітру
 
----------------------------------------------------------------
  VISUAL
----------------------------------------------------------------
 
Hitbox  [B]  [=] [-]
  Розширює хітбокси інших гравців горизонтально.
  Синхронізується з ESP (бокс в ESP збільшується разом).
  · = збільшити на 0.05 блоків
  · - зменшити на 0.05 блоків
  · Без ліміту розширення
 
ESP  [[]
  3D бокс навколо гравців видимий через стіни.
  · Синій напівпрозорий заповнений бокс (~70% opacity)
  · Зовнішнє сяяння (glow)
  · Wireframe контур
  · Синхронізований з Hitbox+
  · Вимикається UnHook і не вмикається поки UnHook активний
  · Ванільні нейм-теги залишаються незмінними
 
----------------------------------------------------------------
  AUTO UTILITIES
----------------------------------------------------------------
 
AutoTotem  [завжди увімкнений]
  Автоматично одягає Тотем Безсмертя в ліву руку:
  · При вході на сервер якщо рука порожня
  · Коли HP падає до 4.5 або нижче
  · Не робить нічого якщо тотем вже в руці
 
AutoAuth  [→]
  При вході на сервер автоматично вводить команду:
    /login IRBBY123
  Спрацьовує один раз через 2 секунди після входу.
  Скидається при відключенні.
 
AutoReg  [↑]
  При вході на сервер автоматично вводить команду:
    /register IRBBY123 IRBBY123
  Спрацьовує один раз через 2 секунди після входу.
  Скидається при відключенні.
 
----------------------------------------------------------------
  MISC
----------------------------------------------------------------
 
NameProtect  [N]
  Замінює твій нік на "Ezz" в чаті та в таб-меню.
  Інші гравці і адміни бачать твій справжній нік,
  але на твоєму екрані скрізь відображається "Ezz".
 
UnHook  [←]
  Аварійне вимкнення підозрілих функцій.
  Одне натискання — вимикає і ховає:
    TriggerBot, AimBot, TargetStrafe, Hitbox, ESP
  Повторне натискання — повертає все і показує знову.
  Поки UnHook активний — клавіші цих функцій не реагують.
 
================================================================
  HUD (інтерфейс)
================================================================
 
Правий верхній кут:
  Список активних функцій. Зелений = увімкнено, Червоний = вимкнено.
  При UnHook показуються тільки AutoSprint і ShiftTap.
 
Правий нижній кут:
  · Ping — затримка до сервера (мс)  зелений<80 / жовтий<150 / червоний
  · TPS  — тіки на секунду сервера   зелений>18 / жовтий>14 / червоний
  · BPS  — блоків на секунду (тільки при активному TargetStrafe)

================================================================
  ПРИМІТКИ
================================================================
 
  · TargetStrafe використовує setVelocity() для руху.
    На серверах з Grim/Polar може викликати speed флаг.
    Налаштуй SPEED_BOOST якщо потрібно.
 
  · AutoTotem і ElytraSwap використовують client-side
    slot clicks — на деяких серверах може не працювати
    якщо інвентар заблокований античитом.
 
  · NameProtect діє тільки локально (на твоєму екрані).
    Сервер і інші гравці завжди бачать твій справжній нік.
