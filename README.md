# LegitMod — Minecraft 1.21.4 Fabric

## Встановлення
1. Скинь `legitmod-1.0.0.jar` з `build/libs/` в папку `.minecraft/mods/`
2. Потрібен Fabric Loader 0.16.9+ та Fabric API

## Клавіші (призначити в Options → Controls → LegitMod)
| Клавіша | Функція |
|---------|---------|
| **Y**          | TriggerBot вкл/викл |
| **B**          | Hitbox+ вкл/викл |
| **=**          | Hitbox+ збільшити |
| **-**          | Hitbox+ зменшити |
| **I**          | AutoSprint+WTap вкл/викл |
| **H**          | ShiftTap вкл/викл |
| **← (Left)**   | UnHook — сховати Hitbox рядок в HUD |

## Збірка
```
.\gradlew.bat build
```
Готовий jar: `build/libs/legitmod-1.0.0.jar`
